PASOS RAPIDOS EN UBUNTU

1) Descomprime el ZIP.
2) Abre Terminal dentro de la carpeta proyecto_final_openmp.
3) Instala herramientas:
   sudo apt update
   sudo apt install -y build-essential g++ make git python3 python3-pip
   pip3 install pandas matplotlib

   Si pip falla:
   sudo apt install -y python3-pandas python3-matplotlib

4) Recolecta hardware:
   ./scripts/collect_hardware.sh

5) Compila:
   make clean
   make all

6) Prueba rapida para ver que todo funciona:
   W=1280 H=720 ITER=300 RADIUS=7 ./scripts/run_benchmarks.sh
   python3 scripts/plot_results.py

7) Corrida formal recomendada:
   W=3840 H=2160 ITER=1000 RADIUS=15 ./scripts/run_benchmarks.sh
   python3 scripts/plot_results.py

8) Corrida 8K real:
   W=7680 H=4320 ITER=1000 RADIUS=15 ./scripts/run_benchmarks.sh
   python3 scripts/plot_results.py

9) Abre docs/Reporte_Tecnico_OpenMP.docx y completa:
   - enlace de GitHub
   - hardware desde results/hardware_info.txt
   - pega las graficas de results/*.png
   - llena tiempos y scheduler optimo desde los CSV

10) GitHub, con commits claros:
   git init
   git add src/sequential.cpp Makefile README.md
   git commit -m "Commit 1: codigo secuencial base"
   git add src/ai_parallel.cpp
   git commit -m "Commit 2: linea base paralela generada por IA con OpenMP"
   git add src/optimized_openmp.cpp scripts
   git commit -m "Commit 3: optimizaciones manuales de OpenMP"
   git add docs results README.md INSTRUCCIONES_RAPIDAS_UBUNTU.txt
   git commit -m "Commit 4: reporte tecnico y resultados"
